# 🌤️ Weather App

A responsive weather application built using **HTML, CSS, and JavaScript** as part of the **SheCodes Plus Workshop**. The project demonstrates the fundamentals of frontend web development while integrating weather data through a weather API.

---

## 📖 Project Overview

The Weather App allows users to search for a city and view its current weather conditions, including temperature, humidity, wind speed, and a weather icon. It also includes a five-day forecast section to provide users with upcoming weather information.

---

## ✨ Features

- 🔍 Search weather by city
- 🌡️ Display current temperature
- 💧 Display humidity
- 💨 Display wind speed
- 🌤️ Display weather icons
- 📅 Five-day weather forecast
- 📱 Responsive and user-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Weather API
